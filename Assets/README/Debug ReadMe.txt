Использование опций Debug

Для использования Debug опций необхоидмо закинуть префаб на сцену, которую вы хотите продебажить. Также есть необходимость в подстановке значений (например, Canvas)

Для сброса количества пройденных уровней используется "R"
Для перехода по сценам используется верхний ряд цифр.
1-6 прямой переход по уровням
7 - переход к сцене завершения
8 - переход к стартовому комиксу
0 - переход в главное меню

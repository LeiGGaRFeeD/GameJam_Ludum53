Sample Scene v2:
-Зеленоватые треугольники поворачиваются только налево
-Зелёно-елочные треугтольники поворачиваются только направо
-Синие треугольники поворачиваются в обе стороны, в зависимости от нажатия на кнопку
-Розовые треугольники поворачиваются внутрь (левый по часовой, правый против часовой).
 Работает по принципу чётный/нечётный

Также на уровне есть красная палка с радиусом действия, если снаряд попадаёт в её поле,
 то уровень автоматически перезапускается

Ярко-зелёная зона означает финиш
Белый круг с квадратом - старт

Кнопки:
-Кнопка Fire отвечает за выстрел сигнала
-Rotate Left отвечает за поворот налево
-Rotate Right отвечает за поворот направо
-Rotation mixed отвечает за поворот розовых треугольников (сделано так из-за разных скриптов)
